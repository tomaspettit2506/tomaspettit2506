# Guardians of the Chess Grandmaster (Final Year Project)

**Author:** Tomás Pettit  
**Student ID:** G00419414  
**Programme:** BSc (Hons) Software Development, ATU Galway

A web application for playing chess online, built as my Final Year Project. The goal of **Guardians of the Chess Grandmaster** is to provide a smooth, modern chess experience with real-time gameplay and a clean UI.

## Project
- **Repository:** https://github.com/tomaspettit2506/final-year-project

## Key Features (planned / in progress)
- Real-time online chess games
- Player accounts & authentication
- Match history and player profiles
- Responsive UI for desktop and mobile
